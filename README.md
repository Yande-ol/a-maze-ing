## 🚧 Project Status: Under Construction
This repository is under **active development**. The project is being built in modules and will be updated daily as we progress.

## 📝 What are we building?
The goal of this project is to develop a complete maze ecosystem in Python 3.10+, focusing on:

* **Generation Engine:** A modular and reusable package (`mazegen`) to create perfect mazes.
* **Pathfinding:** Implementation of search algorithms to find the shortest path between entry and exit.
* **Visualization:** Maze rendering in terminal mode (ASCII) and/or a graphical interface.
* **Architecture:** Code following strict standards for static typing and linting.
